Squaredle Solver

My latest side project was based on "squaredle" - a word search game where words are found by drawing a path through a grid of letters. Letters get faded out, once all words using them have already been found.




Squardle is fun, but there is nothing quite like ruining the fun by writing a program that finds the words for me!

The solver is written in C# and uses WPF for the UI

The app searches for words in the current puzzle and then displays the path used to construct each word it found.

The user can remove paths they have already tried by clicking and also mark individual letters as faded out. Both of these are used to filter down the paths that will be displayed.
